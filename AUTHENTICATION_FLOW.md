### 1. FirebaseAuthService
**Location**: `FirebaseAuthService.swift`

**Responsibilities**:
- Handle Firebase Authentication
- Manage user sessions
- Fetch user data from Firestore
- Determine user role
- Handle logout

**Key Methods**:
```swift
- login(email, password, completion)
- signUp(email, password, name, role, completion)
- logout(completion)
- resetPassword(email, completion)
```

### 2. ContentView
**Location**: `ContentView.swift`

**Responsibilities**:
- Display login UI
- Collect user credentials
- Call authentication service
- Handle navigation based on role
- Show loading states and errors

### 3. AppState
**Location**: `AppState.swift`

**Responsibilities**:
- Store current user information
- Manage user role
- Handle logout navigation
- Provide user state to entire app

### 4. User Model
**Location**: `User.swift`

**Structure**:
```swift
struct User {
    let email: String
    let password: String  // Not stored in Firestore
    let name: String
    let role: UserRole    // .employee or .client
    let profileImage: String?
}

enum UserRole {
    case employee
    case client
}
```
